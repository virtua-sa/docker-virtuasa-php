# Docker - PHP - php-7.1

> `virtuasa/php:7.1`

This docker container is suitable for PHP 7.1 applications. Features :

* PHP 7.1+
  * Composer 1.4.1+
  * PHPUnit 6.2+
  * PHPUnit 5.7+
* Node.js 6.11+
  * Bower, Gulp, Grunt, Webpack
* Yarn
* Apache 2.4
* Debian 9.1

To customize this setup, changes must only be done to file [`docker-compose.yml`](docker-compose.yml).
